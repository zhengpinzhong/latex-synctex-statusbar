# Changelog

All notable changes to this project will be documented in this file.

## 0.0.5
- Add bilingual UI support for Chinese and English.
- Default tooltip/message language to `auto`, following the VS Code UI language.
- Add English documentation to `README.md`.
- Use `icon.png` as the extension icon for publishing compatibility.

## 0.0.4
- Normalize the extension metadata for public publishing.
- Add repository, license, icon, command contribution, and packaging metadata.
- Add `.gitignore` and `.vscodeignore`.
- Remove committed `.vsix` build artifact from the repository.

## 0.0.3
- Show the SyncTeX status bar button only when a LaTeX/TeX editor is active.
- Prompt the user to install LaTeX Workshop if it is missing.

## 0.0.1
- Initial implementation of the Cursor/VS Code extension.
- Add a status bar button that triggers `latex-workshop.synctex`.
